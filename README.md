# ☕ Blaziken Coffee - E-commerce Platform

A full-stack e-commerce platform for a coffee brand, built with modern web technologies.  
This project focuses on scalability, performance, and a seamless user experience.

---

## 🚀 Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **Backend / CMS:** Sanity CMS (Headless CMS)  
- **Database:** PostgreSQL  
- **Deployment:** Vercel  

---

## ✨ Features

- 🛒 Dynamic product catalog managed via Sanity CMS  
- 🔍 Advanced filtering system (origin, flavor profile, etc.)  
- 🔗 Dynamic routing using slugs  
- 📦 Product availability tracking  
- ⚡ Optimized assets for better performance  
- 📱 Fully responsive design  

---

## 🧠 Architecture

This project follows a **headless architecture**, decoupling the frontend from the content management system.

- Content managed in Sanity CMS  
- Data fetched via APIs  
- Scalable and maintainable structure
